# Getting started
1. Install the package through `npm i fire-tracker` or `yarn add fire-tracker`
2. You're done ! Consult the docs and get going !

# Docs
 
 Consult docs [here](https://kokkoooo.github.io)

- No API key required!
- Fast , reliable and flexible.
- Easy to use.
- Open source.
- Self-updated.
- Documented.
